# Task 2: Data Visualization

## Problem Statement
The objective of this task is to visualize patterns in the data and compare different features. By creating various graphical representations, we aim to uncover underlying distributions, trends, and relationships within the dataset.

## Dataset Details
* **Dataset Name:** Iris Dataset
* **Description:** A classic dataset containing measurements of sepal length, sepal width, petal length, and petal width for three different species of the Iris flower.

## Approach
The visualizations were developed in a Jupyter Notebook using Python's data visualization libraries (`matplotlib` and `seaborn`). To fully explore and compare the features, the following plots were generated:
* **Bar Chart (`bar_chart.png`):** Created to compare aggregate feature values across different categories.
* **Histogram (`histogram.png`):** Plotted to visualize the frequency distribution of individual feature measurements.
* **Scatter Plot (`scatter_plot.png`):** Generated to examine the relationship and potential clustering between two specific features.
* **Pairplot (`pairplot.jpg`):** Utilized to comprehensively compare all pairwise relationships across the dataset simultaneously.
* **Heatmap (`heatmap.png`):** Implemented to display the correlation matrix, highlighting the strength of linear relationships between all numerical features.

## Results
The task yielded clear and meaningful visualizations that successfully highlight distinct patterns within the dataset. The generated plots effectively demonstrate how different Iris species can be distinguished based on their sepal and petal dimensions, fulfilling the objective of feature comparison.
